# Plan 02-03: Agent — PTZ Joystick Movement Commands

## Objective

Add PTZ joystick movement commands to the local agent:
1. `ptz_move` — continuous pan/tilt in a direction at a given speed (1-100)
2. `ptz_zoom` — zoom in/out at a given speed (1-100)
3. `ptz_stop` — immediately halt all PTZ movement

These commands use the BirdDog REST API (same as Phase 1 preset recall). The agent's router dispatches them with the same ACK protocol used for all other commands.

## Prerequisites

- Phase 1 agent complete (tunnel, router, OBS control, PTZ preset recall/save)
- PTZ module (`agent/src/ptz.js`) already has REST API communication pattern
- Router already has ACK protocol with requestId tracking
