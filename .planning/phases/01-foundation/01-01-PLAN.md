  <action>
    **Step 1 — Tunnel Module (`cloud/server/src/ws/tunnel.ts`):**
    
    Create and export:
    - `setupWebSocket(server)` — Creates `WebSocketServer({ noServer: true })`, attaches upgrade handler to HTTP server. On path `/ws`: authenticates agent via token in query param `?token=<AGENT_SHARED_SECRET>` (agent sends shared secret, not JWT — it's a machine, not a user). On path `/frontend`: authenticates via JWT in `?token=<jwt>` (frontend clients send JWT). All other paths: destroy socket.
    - `sendToAgent(requestId, command)` — Sends JSON `{ type: 'command', requestId, command, timestamp: Date.now() }` to the active agent. If no agent connected, returns error. Track pending commands: store `{ requestId, command, timestamp, status: 'pending' }` in a Map.
    - `broadcastToFrontend(message)` — Sends JSON to all connected frontend WebSocket clients.
    - `getAgentState()` — Returns `{ connected: boolean, lastHeartbeat: number, health: { obs, ptz } }`.
    
    **Agent message handling:**
    - `{ type: 'ack', requestId, status, error? }` — Resolve pending command. Remove from pending Map. Broadcast `{ type: 'command_result', requestId, status }` to frontend clients.
    - `{ type: 'health', subsystems: { obs, ptz } }` — Update agent state. Broadcast `{ type: 'agent_health', subsystems }` to frontend clients.
    
    **Stale command rejection:** Periodically (every 5s) check pending Map. Remove commands older than 15s. Broadcast `{ type: 'command_result', requestId, status: 'stale' }` to frontend.

    **Step 2 — Server Entry Point (`cloud/server/src/index.ts`):**
    Wire everything together:
    ```typescript
    import express from 'express';
    import { createServer } from 'http';
    import { setupWebSocket } from './ws/tunnel.js';
    import authRouter from './routes/auth.js';
    import commandsRouter from './routes/commands.js';
    import { authenticateToken } from './middleware/auth.js';
    
    const app = express();
    app.use(express.json());
    
    // Public routes
    app.use('/api/auth', authRouter);
    
    // Protected routes
    app.use('/api', authenticateToken, commandsRouter);
    
    const server = createServer(app);
    setupWebSocket(server);
    
    const PORT = process.env.PORT || 3000;
    server.listen(PORT, () => console.log(`CamFlow cloud running on port ${PORT}`));
    ```
    
    Package.json with `"type": "module"`, scripts for `dev` (tsx watch), `build` (tsc), `start` (node dist/index.js).
    
    Railway deployment: Use Railway's Node.js builder. `start` command: `npm start`. SQLite DB path: `./data/camflow.db` — Railway volume mounted at `/data`.

Generate `requestId = crypto.randomUUID()`. Call `sendToAgent(requestId, { type: 'obs_scene', sceneName })`. Store pending command. Return 202 `{ requestId, status: 'pending' }`.

    **PTZ Endpoints:**
    - `POST /api/ptz/preset/recall` — Body: `{ presetNumber: number }`. Validate 1-8. Generate requestId. Send `{ type: 'ptz_preset_recall', presetNumber }`. Return 202.
